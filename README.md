# FIFA-19-Data-Analysis-and-Visualization-using-Tableau
Built an interactive Tableau dashboard using the FIFA 19 dataset to analyze player performance, market value, wages, and skill distribution. The project uncovers insights on player potential, positional strengths, and club performance through dynamic visualizations and filters.

# ⚽ FIFA 19 Player Insights Dashboard (Tableau)

## 📌 Introduction
FIFA is one of the most popular football video games developed by EA Sports. It simulates real-life football matches using real-world players, each having detailed attributes such as speed, shooting accuracy, and defensive skills.

This project analyzes the **FIFA 19 dataset** using **Tableau** to uncover insights into player performance, market value, skills, and potential.

---

## 📊 Dataset Overview
The dataset contains comprehensive information about football players featured in FIFA 19.

### 🔑 Key Features
- Player demographics (Name, Age, Nationality)
- Performance metrics (Overall, Potential)
- Financial data (Market Value, Wage)
- Positional data (Position, Club)
- Skill attributes (Ball Control, Passing, Shooting, Goalkeeping)

---

## 🎯 Project Objective
To build an **interactive Tableau dashboard** that analyzes:
- Player performance trends  
- Market value vs wage relationships  
- Skill distribution across positions  
- Player growth potential  

---

## 📌 Key Analysis Tasks

### 1. Player Performance Analysis
- Compare overall ratings by nationality, age, and position  
- **Visualization:** Bar chart / Heatmap  

### 2. Market Value vs Wage
- Analyze correlation between player value and wages  
- **Visualization:** Scatter plot  

### 3. Player Skill Analysis
- Compare skills across positions (e.g., ball control, shooting)  
- **Visualization:** Bar chart / Box plot  

### 4. Potential vs Overall Rating
- Evaluate future potential vs current performance  
- **Visualization:** Scatter plot  

### 5. Age Distribution
- Analyze age across body types and work rates  
- **Visualization:** Histogram / Pie chart  

### 6. Reputation vs Weak Foot
- Study correlation between reputation and weak foot  
- **Visualization:** Bubble chart  

### 7. Contract Analysis
- Analyze contract length vs player potential  
- **Visualization:** Bar chart  

### 8. Height vs Weight Distribution
- Explore physical attributes by position/body type  
- **Visualization:** Scatter plot  

### 9. Top Players by Market Value
- Identify highest-valued players  
- **Visualization:** Bar chart / Treemap  

### 10. Top Performing Clubs
- Analyze clubs with highest average ratings  
- **Visualization:** Bar chart + Age distribution
